# SynMind Design System

## Description
Design System completo para o ecossistema SynMind - uma plataforma de inteligência de negócios através de pessoas. O sistema abrange três marcas: SynMind (B2B institucional), I_DEAL by SynMind (B2C comercial), e MindScan (base científica), cada uma com identidade visual e tom de voz distintos.

## Problems & Solutions

### Problem 1: Inconsistência Visual Entre Marcas
As três marcas do ecossistema (SynMind, I_DEAL, MindScan) precisam de identidade visual coesa mas diferenciada. O design system estabelece tokens compartilhados com variações semânticas para cada contexto de marca.

### Problem 2: Dualidade B2B/B2C
A comunicação B2B (executiva, consultiva) e B2C (empática, acolhedora) exigem componentes adaptáveis. O sistema oferece variantes de componentes que refletem o tom correto para cada audiência.

### Problem 3: Escalabilidade de Produtos
Novos produtos seguem a convenção de nomenclatura (sufixo "Mind" para B2B) e precisam herdar a identidade visual. O design system fornece padrões reutilizáveis para expansão consistente.

### Problem 4: Credibilidade Científica
A marca MindScan precisa transmitir validação científica e precisão. O sistema inclui selos, badges e componentes específicos para comunicação técnica.

## Key Features
- Tokens de cor baseados na paleta de marca (Azul Corporativo, Laranja Energia, Neutros)
- Tipografia hierárquica para B2B (executiva) e B2C (humanizada)
- Sistema de componentes com variantes por marca/contexto
- Iconografia baseada nos elementos geométricos do logo (arcos, círculos)
- Selos e badges de credibilidade ("Powered by MindScan")
- Suporte a light/dark mode
- Responsividade mobile-first
- Acessibilidade WCAG 2.1 AA
